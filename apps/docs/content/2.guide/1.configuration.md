# Configuration

Learn how to configure Blink for your specific use case.

## Environment Variables

Blink can be configured using environment variables. Create a `.env` file in the `apps/web` directory:

```env [apps/web/.env]
# Server Configuration
NUXT_HOST=0.0.0.0
NUXT_PORT=3000

# WebRTC Configuration
NUXT_PUBLIC_PEER_HOST=localhost
NUXT_PUBLIC_PEER_PORT=9000

# Socket.io Configuration
NUXT_PUBLIC_SOCKET_PATH=/socket.io
```

## Configure Network Discovery

### Change the signaling server port

By default, Blink uses port 9000 for PeerJS signaling. To change this:

```env
NUXT_PUBLIC_PEER_PORT=8080
```

### Configure external access

If you're deploying Blink to be accessible outside your local network:

```env
NUXT_PUBLIC_PEER_HOST=your-domain.com
NUXT_PUBLIC_PEER_SECURE=true
```

## Customize the UI Theme

Blink uses Tailwind CSS with a custom synthwave theme. To customize colors:

1. Navigate to the shared config package:
```bash
cd packages/config
```

2. Edit `tailwind.config.ts`:
```typescript
export default {
  theme: {
    extend: {
      colors: {
        primary: {
          // Your custom colors
        }
      }
    }
  }
}
```

3. Rebuild the application:
```bash
npm run build
```

## Configure File Transfer Settings

File transfer behavior can be customized in `apps/web/app/composables/useFileTransfer.ts`:

```typescript
// Chunk size for file transfers (in bytes)
const CHUNK_SIZE = 16384 // 16KB

// Maximum concurrent transfers
const MAX_CONCURRENT_TRANSFERS = 3
```

::warning
Larger chunk sizes may improve transfer speed but can cause issues on slower networks or devices.
::

## Turborepo Configuration

The monorepo build system is configured in `turbo.json`:

```json [turbo.json]
{
  "tasks": {
    "dev": {
      "cache": false,
      "persistent": true
    },
    "build": {
      "dependsOn": ["^build"],
      "outputs": [".nuxt/**", ".output/**"]
    },
    "typecheck": {
      "dependsOn": ["^typecheck"]
    }
  }
}
```

### Modify caching behavior

To change which tasks are cached:

```json
{
  "tasks": {
    "build": {
      "cache": true,  // Enable caching
      "outputs": [".nuxt/**", ".output/**"]
    }
  }
}
```
