<task type="auto">
  <name>Task 1: Create design system CSS with typography and animations</name>
  <files>app/assets/css/design-system.css</files>
  <action>
    Create a design system CSS file with:

    1. **Typography scale** (Linear-inspired):
       - Font: system-ui with -apple-system fallback (already in app.vue, keep consistent)
       - Heading hierarchy: h1 (2rem bold), h2 (1.5rem semibold), h3 (1.25rem medium)
       - Body: 1rem/1.5 line-height
       - Small: 0.875rem for secondary text
       - Use CSS variables for easy theming

    2. **Spacing scale** (8px base):
       - --space-1: 0.25rem (4px)
       - --space-2: 0.5rem (8px)
       - --space-3: 0.75rem (12px)
       - --space-4: 1rem (16px)
       - --space-6: 1.5rem (24px)
       - --space-8: 2rem (32px)

    3. **Animation primitives** (subtle, Linear-like):
       - --transition-fast: 150ms ease
       - --transition-base: 200ms ease
       - --transition-slow: 300ms ease
       - Keyframes: fadeIn, slideUp, scaleIn (for cards appearing)
       - Keyframes: pulse-subtle (for status indicators, less aggressive than current)

    4. **Interactive states**:
       - Hover lift effect: translateY(-2px) with shadow increase
       - Active press: translateY(0) scale(0.98)
       - Focus ring: 2px offset orange outline for accessibility

    5. **Card base styles**:
       - .card-base: rounded-xl, subtle border, shadow, hover transition
       - .card-interactive: extends card-base with hover/active states
  </action>
  <verify>File exists at app/assets/css/design-system.css with all CSS variables and keyframes defined</verify>
  <done>Design system CSS created with typography, spacing, animations, and card primitives</done>
</task>
